# SC4_lib

Simple tools for working with ScintPi data.

This repository contains small utilities and pipelines for:

- converting raw data to Parquet
- adding derived products
- running lightweight processing workflows

## Project Layout

- `src/scintpi_tools/io/` - file reading utilities
- `src/scintpi_tools/services/` - core processing steps
- `src/scintpi_tools/pipelines/` - higher-level pipeline scripts
- `tests/` - test and example scripts
